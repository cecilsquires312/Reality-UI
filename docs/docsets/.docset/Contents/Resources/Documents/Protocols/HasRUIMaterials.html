HasRUIMaterials

public protocol HasRUIMaterials : HasRUI

An interface used for RealityUI entities that have materials generated from colours These materials change with ruiEnabled and self.RUI.respondsToLighting.

  • updateMaterials()

    All RealityUI Entities should have a method for updating all the materials This is in case of disabling entities or changing their responsiveness to light. This method does not need to be called by outside of a RealityUI class.
